Projeto: Sistema de Gestão de Acervo Didático (NEPISAES)

Desenvolvi e implementei um Sistema de Informação com Banco de Dados Relacional para otimizar a gestão do acervo físico de livros didáticos no cursinho pré-vestibular público NEPISAES.

Desafio e Solução

O controle de doações e a circulação dos livros era feito de maneira informal, resultando em dificuldades de acesso e organização. A solução consiste em um sistema digital que centraliza o gerenciamento, permitindo que os estudantes consultem a disponibilidade de obras em tempo real, registrem empréstimos e devoluções, e garantam a máxima circulação do acervo.

Tecnologias Utilizadas

Banco de Dados: MySQL Workbench (design e estruturação do banco de dados relacional).

Backend / Interface Web: Python com Flask (desenvolvimento da interface inicial).

Atualmente, a interface está operacional para acesso via rede local, e a próxima fase de desenvolvimento, com conclusão prevista para janeiro de 2026, incluirá a implantação final e melhorias na escalabilidade.
